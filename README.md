| :----: | :-------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------ | ----------------------------------------------------------- |
|    0   | `0/`                              | `0/Saliendo...`                                                                                                                                  | Desconexión. El cliente solicita cerrar la conexión.                                        |                                                             |
|    1   | `1/{nombre}/{email}/{contraseña}` | `1/[*] Usuario creado con exito` <br> o <br> `1/ERROR: el usuario ya existe`                                                                     | Registro de un nuevo usuario. El servidor verifica si existe y lo crea.                     |                                                             |
|    2   | `2/{nombre}/{contraseña}`         | `2/1` <br> o <br> `2/ERROR: Contrasena o usuario incorrecto`                                                                                     | Inicio de sesión. Se autentica al usuario.                                                  |                                                             |
|    3   | `3/`                              | `3/Usuarios: {lista_de_usuarios}` <br> o <br> `3/ERROR: No hay Usuarios`                                                                         | Solicitud de la lista de todos los usuarios registrados.                                    |                                                             |
|    4   | `4/`                              | `4/El Top1 es: {nombre}` <br> o <br> `4/ERROR: No hay Usuarios`                                                                                  | Solicitud del usuario con mayor saldo.                                                      |                                                             |
|    5   | `5/`                              | `5/La lista es: {lista_de_conectados}`                                                                                                           | Solicitud de la lista de usuarios actualmente conectados.                                   |                                                             |
|    6   | `6/{nombre_jugador}/{numeroSala}` | `8/Solicitud enviada` <br> o <br> `100/Error: Jugador {nombre} no encontrado` <br> o <br> `100/Error al enviar solicitud a {nombre}`             | Invitación para jugar: el cliente invita a otro usuario.                                    |                                                             |
|    7   | \`7/{1                            | 0}/{nombre\_jugador}\`                                                                                                                           | `8/Invitacion Aceptada` <br> o <br> `8/Invitacion Denegada` <br> o <br> `100/Error formato` | Respuesta a una invitación: 1 para aceptar, 0 para denegar. |
|    8   | `8/{numeroSala}`                  | `10/{numeroSala}/{nueva_capacidad}` <br> o <br> `1/ERROR, Saldo insuficiente...` <br> o <br> `1/Error capacidad llena de Sala...`                | Unirse a una sala. El servidor resta el buy‑in y añade al jugador.                          |                                                             |
|    9   | `9/{numeroSala}`                  | `12/{numeroSala}/{número_jugadores}`                                                                                                             | Salida de la sala. El servidor actualiza el estado de la sala.                              |                                                             |
|   10   | `10/{mensaje}/{numeroSala}`       | `1/{tuNombre}: {mensaje}/{numeroSala}`                                                                                                           | Envío de un mensaje de chat dentro de la sala.                                              |                                                             |
|   11   | `11/{contraseña}`                 | `1/[!] Usuario Eliminado con exito` <br> o <br> `1/ERROR: Contraseña incorrecta`                                                                 | Baja del usuario (elimina su cuenta).                                                       |                                                             |
|   12   | `12/{numeroSala}`                 | `1/La sala {n} tiene suficientes jugadores para iniciar la partida` <br> o <br> `1/La sala {n} no tiene suficientes jugadores...`                | Iniciar partida en sala. Reparte cartas y notifica a los jugadores.                         |                                                             |
|   13   | `13/{fechaMin}/{fechaMax}`        | `16/0/No se pudo obtener el historial` <br> o <br> `16/1/Sin partidas en ese periodo` <br> o <br> `16/2/{lista_de_partidas}`                     | Consultar historial de partidas entre dos fechas para el usuario.                           |                                                             |
|   14   | `14/{nombre_jugador}`             | `17/0/No se pudo obtener el historial` <br> o <br> `17/1/Sin partidas jugadas con {jugador}` <br> o <br> `17/2/{historial}`                      | Historial de partidas jugadas con otro jugador concreto.                                    |                                                             |
|   15   | `15/`                             | `18/0/No se pudo obtener la lista de jugadores` <br> o <br> `18/1/No has jugado con ningún otro jugador` <br> o <br> `18/2/{lista_de_jugadores}` | Lista de todos los jugadores con los que has jugado alguna vez.                             |                                                             |
|   16   | `16/{numeroSala}`                 | `19/{numeroSala}/{jug1:card1,card2;jug2:...;}`                                                                                                   | Mostrar mano de cada jugador en la sala (revelación de cartas).                             |                                                             |
|   17   | `17/{numeroSala}`                 | `22/{numeroSala}/{ganador}/{premio}`                                                                                                             | Fin de partida: anuncia ganador, actualiza saldos y limpia la sala.                         |                                                             |
